Retail Customer Behavior Analysis

Project Overview
This project analyzes retail customer shopping behavior to identify trends, purchasing patterns, and sales insights.

The analysis is performed using Python, SQL, and Power BI.

Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- SQL
- Power BI
- Excel

Dataset
Retail customer transactions dataset containing information about:
- Customer demographics
- Product categories
- Purchase amounts
- Payment methods
- Seasonal trends

Project Workflow
1. Data Cleaning using Python
2. Exploratory Data Analysis
3. SQL queries for business insights
4. Interactive dashboard creation using Power BI

Key Insights
- Identified top selling product categories
- Analyzed customer spending patterns
- Observed seasonal sales trends
- Compared spending behavior by gender and age

Dashboard
Power BI dashboard visualizes:
- Total revenue
- Sales by category
- Seasonal trends
- Customer demographics
